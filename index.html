Hexo博客搭建

2018-08-23

本文主要针对mac OS
hexo博客就是你现在打开的这种

  1. 创建Github 域名和空间

    1.1注册Github账号
    1.2 创建仓库

    创建仓库的名字为name.github.io 
其中的name 要与用户名一致
name下面会用到

  2. 安装

    2.1安装Git
    $ brew install git // 安装Git
    也可以通过下载安装程序来安装

    2.2安装Nodejs
    上网搜一下怎么安装node全是

    2.3安装Hexo
    以上所有都安装完成之后安装Hexo
    $ sudo npm install hexo-cli -g

  3. 编写，发布

    $ hexo init name.github.io
    注意这里用到了name

    安装主题
    $ cd name.github.io //这里是进入到刚才新建的目录
    $ git clone https://github.com/Suolawangzai/Sw-blog //这里的主题就是我这个的主题，其他的主题到官网查看有很多

    基础配置：打开文件夹name.github.io下的_config.yml修改几个地方，下面把几个必须设置的列出来按需求修改，记得保存， 还有注意配置的键值之间一定要有空格。
    title: 来自魔王阎 //你博客的名字
    author: Yan //你的名字
    language: zh-Hans //语言 中文
    theme: Sw-blog //刚刚安装的主题名称
    type: git //使用Git 发布
    repo: https://github.com/name/name.github.io.git // 刚创建的Github仓库 这里可能要自己添加 添加到type下面就行

    添加文章内容
    在name.github.io/source/_posts添加后缀名为md的文件
    如：

    -！-！-  （！号去掉）
title: hello world
---
> 这里就是一个例子。

    测试
    $ hexo s //https://localhost:4000 访问。

    安装hexo-deployer-git自动部署发布工具
    $ npm install hexo-deployer-git –save

    发布
    $ hexo clean && hexo g && hexo d
    第一次上传需要输入Github 邮箱和密码
    以后在每次把博客写完后，执行一下这个命令就可以直接发布了

    博客已经完成，在浏览器中输入 http://name.github.io 就能够访问了。

    更改主题
    只要吧themes目录下的文件删掉 gitclone想要的主题就好，这里别忘了更改上面的配置
